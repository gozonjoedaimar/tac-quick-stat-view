View/Compare your favorite TAC units

Compare your TAC units in bulk using this simple code on your DESKTOP.

Get the code here

Just paste that code

1.) Navigate and filter units in http://alchemistcodedb.com/units.
2.) Open console (F12) in browser.
3.) Paste the code and hit enter button.
4.) Type tacCompareSummary(60) and hit that enter button to see the following.

Note

In this example, Units were filtered by Ninja Job and currently set to LVL 60 (i.e. "base (60)").

This calculation does not include Job Stat Modifier % like the Unit Planner had.

For example, Yomi has a total of 9% patk job stat modifier for Ninja Job.

In Unit planner she will have appx. 380 patk at LVL 60
